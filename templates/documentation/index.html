How To Markup Texts Manually?

After you have set up your Github account and fetched the file from the repository you will notice that all texts are stored in XML files. The content of the files is follows the standards of the Text Encoding Initiative. A short introduction to the topic can be found here: http://www.teibyexample.org/. If you intend to contribute your own text the best method is to open one of the texts from our repositry and fill the standardized structure of the TEI document with the content.

To provide the reader on the website with visual aids for a quicker understanding of the texts the following semantic markups are currently supported: place names, personal names, dates, and terms. The respective tags are <persName>, <placeName>, <date>, and <term>. Characters marked up with these texts appear later coloured on the website and can be clicked on the get further information.

How To Markup Texts Automatically?

Use the UIMA analysis pipeline. Either the online version on this website or the standalone version at our developer site.
