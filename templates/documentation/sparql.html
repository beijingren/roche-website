SPARQL

SPARQL (SPARQL Protocol And RDF Query Language) provides languages and protocols to query and manipulate RDF (Resource Description Framework) graphs. Check out the specification or the book "Learning SPARQL" by Bob DuCharme to learn more. We run a local public SPARQL endpoint you can query directly through a web interface. Currently we use Fuseki as our SPARQL server implementation.

For example if you want query for all female persons from the Qing dynasty those died over the age of 50, paste the following into the our endpoint.

OWL
