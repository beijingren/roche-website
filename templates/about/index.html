About

Welcome to the website of the Siku quanshu project. The objective of the website is to provide an open source environment for collaborative annotation of classical Chinese texts.

All content is provided under a Creative Commons license to allow the free usage of texts for academic purposes. Following international standards in digital humanities the texts are marked up according to Text Encoding Initiative (TEI) (http://www.tei-c.org) guidelines and stored in an XML format, which facilitates the usage of texts in other projects. The open source framework of the website itself encourages the implementation of toolsets for text analysis on the website.

The website offers the following key features:

  • service for Optical Text Recognition (OCR)
  • search within texts
  • simple annotation editor
  • automated markup

This project was funded by the Karl Steinbuch Foundation.

For more information on contributing texts please checkout the documentation.
