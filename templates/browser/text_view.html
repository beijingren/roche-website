  • Help and Shortcuts

    In this text view you can annotate expressions. Mark a section of the text and press a key to send the selection to the server. Press 1 for person names, 3 for place names and t for terms.

    There is no need to wait for the server response. Annotations are send to a work queue and will be processed in sequence. But the server response will cause a page reload to show the new annotation, so please reload the page yourself after a few second, if you annote a text in fast sequence.

    The UIMA worker will try to annotate the juan the annotation you selected occurs in first to provide a fast response. The whole work is then processed afterwards.

    Thanks for your contribution!
